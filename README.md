# Iris Flower Classification

This project is part of the CodeAlpha Data Science Internship.

## Overview
The objective of this project is to classify Iris flowers into three different species
based on flower measurements using machine learning.

## Dataset
The Iris dataset includes features such as:
- Sepal length
- Sepal width
- Petal length
- Petal width

## Model Used
Logistic Regression

## Tools & Libraries
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Seaborn

## Result
The model achieved an accuracy of around 95%.
